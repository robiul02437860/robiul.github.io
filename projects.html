RPO Trajectory Visualizer

Developed an interactive 3D visualization tool for satellite rendezvous and proximity operations (RPO) using PyVista. The tool plots predicted particle trajectories with color-coded uncertainty over time, aiding mission planning and analysis.

  • Tech Used: PyVista, NumPy, Python
  • Dataset: RPO mission planning data
  • Outcome: Improved interpretability for non-technical stakeholders
